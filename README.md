# 人脸签到系统 (Facial Recognition Attendance System)

## 项目概述

本项目是一套完整的人脸识别签到系统，支持手机端拍照上传进行签到，服务器端完成人脸识别与身份匹配。系统基于华为云FRS服务实现核心人脸识别功能，部署在华为云ECS服务器上。

## 快速开始

### 后端部署

```bash
# 1. 克隆项目
git clone https://github.com/Wander-C/facial-recognition-attendance.git
cd facial-recognition-attendance

# 2. 创建Python虚拟环境
python3.10 -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# 3. 安装依赖
pip install -r backend/requirements.txt

# 4. 配置环境变量
cp .env .env
# 编辑 .env 文件，填入数据库、Redis、华为云配置

# 5. 初始化数据库
cd backend
python init_db.py

# 6. 启动服务
uvicorn main:app --host 0.0.0.0 --port 8000
```

## 核心功能

### 用户模块
- 用户注册、登录、注销
- JWT Token维护会话
- 密码加密存储

### 人脸库模块
- 用户注册时将人脸照片添加到FRS人脸库
- 支持人脸库管理
- 人脸ID与用户ID关联

### 签到模块
- 接收签到照片
- 调用FRS人脸搜索接口识别用户
- 相似度阈值验证（建议80%）
- 签到限流处理

### 记录模块
- 查询个人签到记录
- 支持按时间筛选
- 签到统计分析
- 管理员查看所有记录

## 高并发处理

- **限流**: 同一用户5秒内不能重复签到，防止恶意请求
- **削峰**: 使用Redis缓存减少数据库写入
- **缓存**: 缓存当日已签到用户信息
- **重试**: FRS调用失败时自动重试
