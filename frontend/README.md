# 前端开发指南

## 项目概述

人脸签到系统的前端为手机应用，支持以下功能：
- 用户注册和登录
- 人脸照片上传
- 拍照签到
- 签到记录查看

## 技术栈

- **开发框架**: uni-app 或 原生Android
- **网络请求**: Axios
- **图片处理**: 相机API + 图片压缩库

## 项目结构

```
frontend/
├── pages/
│   ├── login/              # 登录页面
│   ├── register/           # 注册页面
│   ├── home/               # 主页
│   ├── sign_in/            # 签到页面
│   └── records/            # 记录查看页面
├── components/             # 组件
├── utils/
│   ├── api.js              # API请求
│   └── storage.js          # 本地存储
├── static/                 # 静态资源
└── App.vue                 # 应用入口
```

## 开发步骤

### 1. 用户认证模块
- 实现注册页面
- 实现登录页面
- JWT Token管理
- 登出功能

### 2. 个人信息模块
- 显示用户信息
- 人脸照片上传
- 人脸库管理

### 3. 签到模块
- 拍照功能
- 图片压缩
- 上传签到
- 实时反馈

### 4. 记录查看模块
- 签到记录列表
- 记录筛选
- 统计信息

## 常见问题

- **图片上传失败**: 检查网络和服务器连接
- **人脸识别失败**: 确保照片清晰、光线充足
- **Token过期**: 自动刷新或重新登录
